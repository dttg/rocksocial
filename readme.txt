Seu plugin de botões de compartilhamento, o RockSocial é simples e fácil de usar. Baseado no plugin Digg Digg, mas com muito mais facilidade de uso. 

== Descrição ==
Com o RockSocial você tem botões de compartilhamento para as principais redes sociais no seu blog. Exiba todos os seus perfis de redes sociais de modo fácil e lindo! 

= Funcionalidades =

* Exiba os principais botões de compartilhamento com contador. Twitter, Facebook, Linkedin, Google +1, Pinterest, Tumblr.

* Suporte a múltiplos idiomas
* Simples e fácil de configurar. Escolha uma barra flutuante como esta: http://marketingdeconteudo.com/rocksocial ou botões no topo ou no final de cada post
* Precisa de alguma melhoria? Fale conosco: contato@rockcontent.com

== Instalação ==

1. Baixe o plugin e descomprima sua pasta
2. Copie a pasta rocksocial para a pasta "/wp-content/plugins/" dentro do seu Wordpdress
3. Ative o plugin no menu 'Plugins' do WordPress
4. Estamos prontos – Para customizar o plugin, vá até o menu "Rocksocial" na área de configurações do seu blog.
